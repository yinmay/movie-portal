# Movie Portal

The OMDv for fetch movie data.

## Installation

- git clone https://github.com/yinmay/star_war.git
- `yarn` or `npm install`

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## project structure

```
src
   components
      Breadcrumbs
      FilterTable

  Pages

   util

```
